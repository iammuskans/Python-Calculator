# Python Calculator

A simple command-line calculator built using Python.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Error handling using try-except
- Continuous loop calculator

## Technologies Used
- Python
- Git
- GitHub

## How to Run

1. Clone the repository

git clone https://github.com/iammuskans/Python-Calculator.git

2. Navigate to the folder

cd Python-Calculator

3. Run the program
